# 手机操作GitHub部署指南

## 📱 您需要完成的步骤（5分钟）

---

### 第1步：在手机上访问GitHub（2分钟）

#### 1.1 下载GitHub App（推荐）

- **iOS**: App Store搜索 "GitHub"
- **Android**: Google Play搜索 "GitHub"

或者直接用浏览器访问：https://github.com

#### 1.2 登录GitHub账户

如果没有账户：
1. 点击 "Sign up"
2. 填写：
   - Username（用户名，例如：zhangsan-stock）
   - Email（你的邮箱）
   - Password（密码）
3. 验证邮箱
4. 完成注册

如果有账户，直接登录即可。

---

### 第2步：创建新仓库（2分钟）

#### 2.1 点击创建仓库

登录后：
1. 点击右上角 **+** 号
2. 选择 **New repository**

#### 2.2 填写仓库信息

填写以下内容：

| 字段 | 填写内容 | 说明 |
|------|---------|------|
| **Repository name** | `stock-monitor` | 仓库名称 |
| **Description** | `薄膜铌酸锂概念股监控` | 描述（可选） |
| **Public/Private** | 选择任意 | Public=公开，Private=私有 |
| **Initialize with README** | ✅ 勾选 | 创建说明文件 |

其他选项：
- ⬜ Add .gitignore（不勾选）
- ⬜ Choose a license（不勾选）

#### 2.3 点击创建

点击绿色的 **Create repository** 按钮

---

### 第3步：告诉我您的GitHub用户名（1分钟）

创建完成后，您会看到仓库地址，格式是：

```
https://github.com/你的用户名/stock-monitor
```

**把您的GitHub用户名告诉我！**

例如：
```
"我的GitHub用户名是：zhangsan-stock"
```

我拿到您的用户名后，会在本地帮您配置git，然后您可以：
1. 在手机浏览器访问仓库
2. 上传文件
3. 完成部署

---

## 📱 备选方案：如果不想用Git

### 方案A：直接在网页上上传文件

1. 进入您的仓库
2. 点击 **Add file** → **Create new file**
3. 复制粘贴代码（我会给您）
4. 填写文件名：`.github/workflows/stock-monitor.yml`
5. 点击 **Commit changes**
6. 重复步骤2-5，创建其他文件：
   - `stock_monitor_simple.py`
   - `requirements.txt`
   - `README.md`

### 方案B：使用GitHub App上传

1. 打开GitHub App
2. 进入您的仓库
3. 点击右下角 **+**
4. 选择 **Upload file**
5. 选择文件上传

3. 找到 "stock-monitor"

点击登录页面的 "Forgot password"，重置密码。
