# Server酱配置指南 - 最简单的微信消息推送

## 🎯 什么是 Server酱？

Server酱是一个将消息推送到微信的工具，**无需注册小程序**，配置非常简单！

---

## 🚀 快速开始（3步搞定）

### 步骤1：获取 SendKey

1. 访问 Server酱官网：https://sctapi.ftqq.com/
2. 点击"微信扫码登录"
3. 登录后，点击"发送消息"
4. 你会看到一个 **SendKey**，复制它（类似：`SCT123456xxxxxxxxxxxx`）

### 步骤2：配置 SendKey

打开 `stock_monitor_tcb_simple.py` 文件，找到第19行：

```python
SCT_SENDKEY = "YOUR_SENDKEY_HERE"
```

替换为你的 SendKey：

```python
SCT_SENDKEY = "SCT123456xxxxxxxxxxxx"  # 替换为你的实际SendKey
```

### 步骤3：部署到微信云开发

1. 访问微信云开发控制台：https://console.cloud.tencent.com/tcb
2. 创建云函数，复制 `stock_monitor_tcb_simple.py` 的内容
3. 配置定时触发器（每5分钟执行一次）

完成！当股价波动超过3%时，你会收到微信消息提醒。

---

## 💡 测试消息推送

部署后，你可以手动触发一次测试：

1. 在云函数控制台，点击"测试运行"
2. 如果有股票波动超过3%，会立即收到微信消息
3. 你也可以临时修改阈值为 `THRESHOLD = 0`，这样所有股票都会触发提醒

测试完成后记得改回 `THRESHOLD = 3.0`

## 🔧 进阶配置

### 修改推送频率

在云函数触发器中修改 Cron表达式：

```yaml
# 每5分钟
0 */5 * * * * *

# 每10分钟
0 */10 * * * * *

# 每小时
0 0 * * * * *

# 交易时间每5分钟（工作日9:30-11:30, 13:00-15:00）
复杂配置，建议使用固定间隔
```

### 修改价格阈值

修改 `stock_monitor_tcb_simple.py` 中的 `THRESHOLD`：

```python
THRESHOLD = 3.0  # 3% 波动触发提醒
# 改为 2.0 则更敏感，5.0 则更保守
```

### 添加更多股票

修改 `STOCKS` 字典：

```python
STOCKS = {
    '600330': {'name': '天通股份', 'category': '上游'},
    '000001': {'name': '平安银行', 'category': '金融'},  # 添加新股票
    # 继续添加...
}
```

## ❓ 常见问题

### Q1: 收不到微信消息
**A**:
1. 确认是否关注了 Server酱公众号
2. 检查 SendKey 是否正确
3. 检查是否有超过阈值的价格波动
4. 查看云函数日志是否有错误

### Q2: 消息发送失败
**A**:
1. 检查云函数是否能访问外网
2. 检查 Server酱服务是否正常
3. 查看云函数日志中的错误信息

### Q3: 如何查看发送记录
**A**:
1. 访问 Server酱官网
2. 登录后可以看到发送历史

### Q4: 可以推送给多个人吗
**A**:
可以！有两个方法：
1. 让他们分别注册 Server酱，获取各自的 SendKey
2. 使用付费版，支持群组推送
