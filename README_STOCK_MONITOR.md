# 股票监控脚本使用指南

## 📋 功能说明

这个脚本可以实时监控薄膜铌酸锂概念股的价格波动，当涨跌幅超过设定阈值时自动发送提醒。

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 配置脚本

打开 `stock_monitor.py`，根据需要修改以下配置：

```python
# 监控的股票列表
STOCKS = {
    '600330': {'name': '天通股份', 'category': '上游'},
    '002222': {'name': '福晶科技', 'category': '上游'},
    # ... 其他股票
}

# 价格波动阈值（百分比）
PRICE_CHANGE_THRESHOLD = 3.0  # 涨跌幅超过3%触发提醒

# 检查间隔（秒）
CHECK_INTERVAL = 300  # 5分钟检查一次
```

### 3. 运行脚本

```bash
python stock_monitor.py
```

脚本会持续运行，每5分钟检查一次股价。

## ⚙️ 配置邮件提醒（可选）

如果需要通过邮件接收提醒，修改以下配置：

```python
EMAIL_CONFIG = {
    'enabled': True,  # 启用邮件通知
    'smtp_server': 'smtp.qq.com',  # QQ邮箱SMTP服务器
    'smtp_port': 587,
    'username': 'your_email@qq.com',
    'password': 'your_password',  # QQ邮箱需要使用授权码
    'from_addr': 'your_email@qq.com',
    'to_addr': 'your_email@qq.com'
}
```

### QQ邮箱授权码获取方法

1. 登录QQ邮箱
2. 设置 -> 账户
3. 开启SMTP服务
4. 生成授权码
5. 将授权码填入 `password` 字段

## 📊 数据来源

脚本使用以下免费API获取股价数据：

- **新浪财经API**: http://hq.sinajas.cn/list=股票代码
  - 实时性较好
  - 无需API密钥
  - 适合个人使用

其他可选的免费API：

1. **腾讯财经API**: http://qt.gtimg.cn/q=股票代码
2. **东方财富API**: http://push2.eastmoney.com/api/qt/stock/get
3. **Yahoo Finance API**: 需要安装 `yfinance` 库

## 📁 输出文件

脚本运行后会生成以下文件：

- `stock_monitor.log`: 运行日志
- `alerts.json`: 历史提醒记录

## 💡 使用建议

1. **检查间隔**: 建议设置为5-10分钟，避免过于频繁
2. **价格阈值**: 建议设置为3%-5%，避免过多噪音
3. **监控时间**: 可以根据交易时间调整，只在交易时段运行
4. **数据可靠性**: 免费API可能有延迟，建议结合多个数据源验证

## 🔧 故障排查

### 问题1: 获取不到股价数据

**解决方法**:
- 检查网络连接
- 尝试更换API源
- 查看日志文件获取详细错误信息

### 问题2: 邮件发送失败

**解决方法**:
- 确认SMTP服务器地址和端口正确
- 使用授权码而非邮箱密码
- 检查防火墙设置

### 问题3: 脚本运行卡顿

**解决方法**:
- 增加检查间隔时间
- 减少监控股票数量
- 优化API请求超时设置
